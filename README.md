# alien_invasion
飞行射击游戏（类似雷电）

## 游戏简介
这是一款飞行射击游戏，在每次新游戏开始时，你的飞船会处于屏幕底部正中央，你被要求利用子弹击杀从屏幕顶部不断飘下的的外星人。

随着你的击杀不断增加，游戏等级会逐渐提高（表现为敌我双方的速度都会加快），同时每击杀一定数量外星人会得到道具奖励（道具的种类是随机的）。

## 按键说明
Space   :   开始新游戏与射击

P       ：  暂停与继续

Left    :   向左移动

Right   :   向右移动

Up      :   向上移动

Down    :   向下移动

1       :   使用道具 1：枪支（延时道具，子弹宽度加倍，可以连续使用）

2       :   使用道具 2：雪花（延时道具，外星人下落速度减半，可连续使用）

3       :   使用道具 3：升级（延时道具，普通子弹升级为高能子弹）

4       :   使用道具 4：闪电（瞬时道具，一键清除所有外星人）

5       :   使用道具 5：时钟（延时道具，加倍其他道具效果的时间，可连续使用）

Pause   :   暂停与继续背景音乐

PgUp    :   上一首背景音乐

PgDn    :   下一首背景音乐

## 游戏设置
每击杀 10 个外星人会提高游戏等级，每击杀 25 个外星人会随机掉落一个道具。

六种道具的掉落几率均等，所有延时道具的效果为 10 秒。

外星人碰到飞船会损失一条生命，同时会有 3 秒无敌时间。

所有外星人作为一个整体进行左右移动，一旦碰到一侧边缘，外星人将整体向另一侧移动。

## 游戏技巧
时钟道具应当在其他道具使用之后的有效时间内使用。

雪花道具一般用于后期，并且配合其他道具效果更好。

闪电道具一般用于后期拾取道具时清除障碍。

枪支道具连续使用效果奇佳，尤其配合时钟之后。
